# GA Estética Automotiva - Landing Page

Projeto em Next.js com Tailwind via CDN, responsivo e estruturado para SEO local.

## Rodar localmente

```bash
npm install
npm run dev
```

Acesse: http://localhost:3000

## Ajustes antes de publicar

1. Troque `https://www.gaesteticaautomotiva.com.br` em `app/layout.js` pelo domínio real.
2. Depois que o domínio estiver no ar, cadastre no Google Search Console.
3. Solicite indexação da página principal.
4. Crie/otimize o Perfil da Empresa no Google com o mesmo nome, telefone e endereço do site.
